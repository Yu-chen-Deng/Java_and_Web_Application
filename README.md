# Environment Setup

+ Apache Tomcat 11.0
+ Smart Tomcat 4.7.3
+ Azul Zulu 17.0.13 (JDK17)
+ IntelliJ IDEA Community Edition 2024.1.4
+ MySQL 8.4
+ Windows 11

# How To Setup
1. 将 swubbs.war 文件部署到 Tomcat 服务器。
2. 在浏览器中访问 Tomcat 指定的网页，格式为 ```http(s):// localhost:端口号 / tomcat设定 / initDB.jsp```，即可完成设置。
